Steps to install xbmc-ball-streams addon:

1. Browse to the download location: 
    > https://github.com/fungus1487/xbmc-hockey-streams/tree/master/release

2. Left click the latest zip file that matches your version of Kodi (XBMC).
    > Eden   = v11
    > Frodo  = v12
    > Gotham = v13
    > Helix  = v14

3. Click 'View Raw' and your browser will prompt you to save locally.

4. Follow these instructions within Kodi (XBMC) under System -> Add-ons: 
    > http://kodi.wiki/view/HOW-TO:Install_an_Add-on_from_a_zip_file
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------
Note:  If the install failed ensure you downloaded the correct version,
       otherwise post error details here for assistance: 
    > http://www6.hockeystreams.com/support/categories/9/general+homebrew/109/XBMC+Plugin
