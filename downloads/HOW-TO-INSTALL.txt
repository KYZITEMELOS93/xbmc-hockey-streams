Steps to install xbmc-hockey-streams addon:

1. Browse to the following location: 
    > https://github.com/fungus1487/xbmc-hockey-streams/blob/master/README.md

2. Click the download link that matches your version of XBMC/Kodi.
    > Eden   = v11
    > Frodo  = v12
    > Gotham = v13
    > Helix  = v14

3. Follow these instructions within Kodi (XBMC) under System -> Add-ons: 
    > http://kodi.wiki/view/HOW-TO:Install_an_Add-on_from_a_zip_file
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------
Note:  If the install failed ensure you downloaded the correct version,
       otherwise post error details here for assistance: 
    > http://www6.hockeystreams.com/support/categories/9/general+homebrew/109/XBMC+Plugin
